1. PAGINA DE APROVACOES (Approvals.jsx)

   Adicionadas bolinhas coloridas nos cards:
   
   - Verde: Aprovada (approved)
   - Amarelo: Necessita Revisão (needs_revision)
   - Vermelho: Recusada (rejected)
   - Cinza: Pendente de Revisão (pending_review)
   
   Caracteristicas:
   - Posicionamento absoluto no canto superior direito
   - Tamanho: 12px (w-3 h-3)
   - Formato: Circular (rounded-full)
   - Sombra: shadow-sm para destaque
   - Tooltip: Mostra status ao passar o mouse

2. FUNCAO HELPER

   Criada funcao getStatusDotColor():
   
   ```javascript
   const getStatusDotColor = (approvalStatus) => {
     switch (approvalStatus) {
       case 'approved':
         return 'bg-green-500'
       case 'needs_revision':
         return 'bg-yellow-500'
       case 'rejected':
         return 'bg-red-500'
       case 'pending_review':
         return 'bg-gray-400'
       default:
         return 'bg-gray-400'
     }
   }
   ```
