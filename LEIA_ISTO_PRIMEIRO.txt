================================================================================
                    🚀 COMO APLICAR A MIGRAÇÃO
================================================================================

OPÇÃO 1 - MAIS RÁPIDA (Recomendada):

1. Feche o servidor Django se estiver rodando (Ctrl+C)

2. Execute o script batch:
   aplicar_migracao.bat

3. Reinicie o servidor:
   cd backend
   python manage.py runserver

4. Pronto! Acesse o sistema e verifique os códigos

--------------------------------------------------------------------------------

OPÇÃO 2 - MANUAL:

1. Aplicar a migration:
   cd backend
   python manage.py migrate core

2. Adicionar códigos às empresas existentes:
   python add_codes_to_existing_companies.py

3. Reiniciar o servidor:
   python manage.py runserver

4. Testar o sistema
