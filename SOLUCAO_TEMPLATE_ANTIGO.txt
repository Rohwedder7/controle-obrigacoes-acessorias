================================================================================
                    ⚠️ PROBLEMA IDENTIFICADO - TEMPLATE ANTIGO
================================================================================

O template que você baixou não tem a coluna "Código" como primeira coluna.
Isso significa que o sistema está servindo o template antigo.

================================================================================
                    🔧 SOLUÇÃO - SIGA ESTES PASSOS
================================================================================

PASSO 1: REINICIAR O SERVIDOR DJANGO

1. Pare o servidor Django (Ctrl+C no terminal onde está rodando)
2. Inicie novamente:
   cd backend
   python manage.py runserver

PASSO 2: LIMPAR CACHE DO NAVEGADOR

1. Pressione Ctrl+Shift+Del
2. Selecione "Imagens e arquivos em cache"
3. Clique em "Limpar dados"

OU

1. Pressione Ctrl+F5 na página para forçar recarregamento

PASSO 3: BAIXAR O TEMPLATE NOVAMENTE

1. Acesse http://localhost:5173
2. Vá em "Empresas"
3. Clique em "Baixar Template"
4. Verifique se agora tem a coluna "Código"

================================================================================
                    📋 VERIFICAÇÃO DO TEMPLATE CORRETO
================================================================================

O template de empresas DEVE ter estas colunas (nesta ordem):

1. Código (obrigatório)
2. Razão Social (obrigatório)
3. CNPJ (opcional)
4. Nome Fantasia (opcional)
5. E-mail (opcional)
6. Telefone (opcional)
7. Endereço (opcional)
8. Responsável (opcional)

Se o template que você baixar NÃO tiver a coluna "Código" como primeira,
significa que o servidor não foi reiniciado ou há cache.
